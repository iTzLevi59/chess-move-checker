# chess-move-checker
Check if chess pieces can move to a specified cell
This is a simple C++ program to check if chess pieces (Knight, Bishop, Rook, Queen, King)
can move to a specified cell on the board.  

## Usage

Compile and run:

```bash
g++ src/main.cpp -o chess
./chess
```
### Input examples:
Start position: C1
Move position: E3
#### Output:
1) The knight cannot move to the specified cell!
2) The bishop can move to the specified cell!
3) The rook cannot move to the specified cell!
4) The queen can move into the specified cell!
5) The king cannot move to the specified cell!
